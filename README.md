# Tech Stack
   - PL   : Back End >> NextJS (15.3.4), Front End >> React Native (0.79.3)
   - UI   : Nativewind (4.1.23), React Native Paper (5.14.5)
   - DB   : PostgreSQL (16.9)


# Setup Project

1. Clone Project
   
   ```bash
   git clone https://github.com/fericopasaribu/Barang.git
   ```
   
3. Install Dependencies

   ```bash
   npm i
   ```

4. Install App

   Android 
   ```bash
   npm run android
   ```

    iOS 
   ```bash
   npm run ios
   ```
